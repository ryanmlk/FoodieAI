# Assets Directory

This directory contains image assets for the FoodieAI app.

## Structure

- `images/` - App images, icons, and illustrations
  - Add your custom images here
  - Supported formats: PNG, JPG, SVG

## Usage

Reference assets in your code like this:

```dart
Image.asset('assets/images/your_image.png')
```
